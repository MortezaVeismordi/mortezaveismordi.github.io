درباره من

توسعه‌دهنده‌ای نتیجه‌گرا با تمرکز روی ساخت وب‌اپلیکیشن‌های مقیاس‌پذیر. تجربه کار با Django/DRF برای backend قوی، ادغام مدل‌های AI/ML با رابط کاربری، بهینه‌سازی دیتابیس و پیاده‌سازی فیچرهای real-time. همیشه به دنبال نوشتن کد تمیز، SOLID و قابل نگهداری هستم.

مهارت‌های کلیدی

Backend

Django • Django REST Framework • Python • PostgreSQL

Frontend

HTML5 • CSS3 • JavaScript • آشنایی با Next.js / React

AI / ML

TensorFlow / PyTorch پایه • پروژه‌های تشخیص بیماری (MRI, Lung)

Tools

Git • Docker • Linux • CI/CD پایه • Redis / Celery
